# ホーム

Lura World Docs は、VRChat 向けに制作したワールドアセットの導入手順とカスタマイズ方法をまとめるためのドキュメントサイトです。

現時点では、以下の 2 ワールドを中心に整理しています。

- [LuminousHotel 解説](./luminous-hotel/overview.md)
- [LuminousOasis 解説](./luminous-oasis/overview.md)

また、今後追加するワールド向けに共通の枠も用意しています。

- [今後のワールド解説](./future-worlds/overview.md)

## このサイトの使い方

各ワールドの元ドキュメントに含まれていた要素を、読みやすい順番で再構成して掲載しています。LuminousHotel は、セットアップ、初期設定、アップロード、改変、トラブル対応の順で追えば、原稿の流れを崩さずに確認できます。

## 現在の構成

### LuminousHotel

- ようこそ
- セットアップと導入
- シーン初期設定
- アップロードと Quest 対応
- トラブルシューティング

### LuminousOasis

- ドキュメント枠を先行整備
- 導入方法と補足解説を今後追加

### 今後のワールド解説

- 新規ワールド追加時の共通テンプレート
- 導入方法とその他解説の受け皿
